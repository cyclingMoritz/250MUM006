University Project: Bicycle Traffic in Barcelona

An in-depth analysis using data from 2017 to 2023

Introduction

This project investigates the potential of permanent inductive loop bicycle counters by analyzing traffic data collected in Barcelona from 2017 to 2023. It explores seasonal patterns, daily trends, and anomalies while emphasizing data cleaning and imputation techniques to improve data reliability.

Visualization 1: Long-Term Trends

This visualization highlights the long-term traffic trends, including a sharp decline during the COVID-19 lockdown and subsequent recovery. The data reflects yearly and overall cycling activity changes over time.

Visualization 2: Weekly and Seasonal Patterns

This plot showcases weekly and seasonal trends, identifying the days and times with the highest cycling activity. Peaks correspond to commuting hours, while seasonal data shows increased activity in warmer months.

Visualization 3: Data Anomalies

This visualization presents anomalies detected in the data, such as unusually high or low counts. It also demonstrates how data cleaning enhanced the accuracy of the analysis.
